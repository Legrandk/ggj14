ALONE IN THE FOREST
===================

  The fate of our hero is at your hands.

  You should help him to defeat dangerous creatures during hist quest for the hidden treasure.

  A marvelous treasure is known to be hidden into a ancient castle.

  Nobody has returned from this journey.


USING THE CONTROLLER
====================

  You can move in any of four directions with the WASD keys in that direct.


LOAD THE GAME
=============

  Open "Alone in the Forest.html" file in a modern HTML5 supported browser.
